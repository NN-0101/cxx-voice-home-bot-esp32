# VoiceHomeBot

以语音为入口，以连接为核心，打造你的私域智能家居控制中枢。

## 🎯 项目简介

VoiceHomeBot 是一个基于 ESP32 的智能语音控制机器人，集成了语音输入（INMP441 麦克风）、语音输出（MAX98357 音频模块）、舵机控制和 OLED 显示功能。可以作为智能家居的中枢，通过语音命令控制家中的各种设备。

| ESP32 引脚 | MAX98357 引脚 | 说明 |
|-----------|--------------|------|
| GPIO27 | BCLK | 位时钟 |

| 5V    | VIN | 电源（5V） |
| GND   | GND | 地线 |
| -     | GAIN | 增益控制（可选） |

### 舵机控制（PWM）

## 🎨 功能特性

### 视觉功能
- **动态表情**：OLED 显示机器人眼睛，支持正常、微笑、闭眼等表情
- **眼球运动**：双舵机控制眼球左右/上下转动，模拟真实目光
- **眨眼动画**：周期性眨眼，让机器人更生动
- **多种表情**：可扩展多种表情库

### 音频功能
- **语音输入**：INMP441 麦克风采集语音信号，支持语音识别
- **语音输出**：MAX98357 功放驱动扬声器，播放语音回复
- **多音效库**：支持多个 WAV 文件存储，可随机播放
- **I2S 数字音频**：高质量音频传输，抗干扰能力强

### 控制系统
- **语音控制**：通过语音命令控制设备和舵机
- **WebSocket 控制**：通过 WebSocket 实现远程控制
- **Wi-Fi 连接**：自动连接 Wi-Fi，支持配网
- **串口调试**：完整的调试信息输出
